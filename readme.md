# Laranat Project

Laranat is a full-stack, cross-platform application designed with scalability, performance, and modern technologies in mind. The project follows a modular architecture to support web, mobile, and backend development independently while sharing a unified vision.

---

## Project Structure

```
Laranat-project/
│
├── client_web/        # React + TypeScript web application
├── client_mobile/     # Flutter mobile application (Android & iOS)
├── server_main_go/    # Go (Golang) main backend service

```

---

## Tech Stack

### Web Client

* React
* TypeScript
* Responsive design (mobile, tablet, desktop)

### Mobile Client

* Flutter
* Cross-platform support (Android & iOS)

### Backend

* Go (Golang) with Gin
* RESTful APIs (future expansion possible)

### Database

* PostgreSQL (primary database)

### Future Enhancements

* Blockchain integration for payments  (team-approved)
* Extended payment modules
* Advanced security layers

---

## Key Principles

* **Responsive Design:** All client applications must adapt seamlessly across devices.
* **Scalable Architecture:** Independent services with clear responsibilities.
* **Cross-Platform First:** Web and mobile developed in parallel.
* **Future-Ready Payments:** Architecture allows blockchain payment integration.

---

## Getting Started

Each major component has its own setup instructions inside its respective folder:

* `client_web/` → Web application setup
* `client_mobile/` → Flutter mobile setup
* `server_main_go/` → Backend service setup

Refer to each directory’s documentation before running the project.
